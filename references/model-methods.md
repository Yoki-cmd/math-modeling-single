### 题型 A · 机理分析类（对应国赛 A 题，难度最高，选人最少）
- **特征**：物理/工程/力学/流体背景，客观规律推导、动态系统、数值仿真，理论深、门槛高
- **关键词**：运动轨迹、力学分析、流体、波动、动态变化、物理仿真
- **核心模型**：常/偏微分方程、数值积分(R-K/ode45)、动力学模型、MATLAB/Simulink 仿真、几何运动学、参数反演
- **适配**：数学/物理/自动化/机械（高数+工程力学基础）
- **AI 边界**：AI 仅辅助公式推导核对、仿真代码排错；**机理与方程建立必须人工**
- **→ 进 A1 微分方程 / A2 几何运动学 / A3 反问题反演；常配 U1 蒙特卡洛、B3 元启发式（几何优化）**

### 题型 B · 综合优化类（对应国赛 B 题，综合性最强）
- **特征**：开放度高，图论+运筹+规划+调度，场景多变(无人机/路线/调配)，无固定套路，建模+算法并重
- **关键词**：调度、路径、分配、编队、最优方案、约束条件
- **核心模型**：线性/非线性规划、多目标规划、图论(Dijkstra/最小生成树/最大流)、0-1规划、遗传/退火/A*
- **适配**：计算机/运筹/经管/电气（逻辑梳理+算法调试）
- **AI 边界**：AI 可荐算法、写基础代码；**约束设置、目标函数调整、多方案对比靠人工**
- **→ 进 B1 规划 / B2 图论 / B3 启发式；大规模整数规划见 D1 求解器选择要点**

### 题型 C · 数据分析类（对应国赛 C 题，选人最多，零基础友好）
- **特征**：经管/统计/生活场景，附大量原始数据，核心是清洗+挖掘+统计分析，门槛低、拿高分靠分析与论文出彩
- **关键词**：数据统计、相关性、分类、聚类、趋势、特征分析、显著性检验
- **核心模型**：回归、K-Means/层次/DBSCAN、时间序列、BP/神经网络、评价(AHP/熵权/TOPSIS)、统计检验、SPSS/Python(Pandas)
- **适配**：经管/统计/大数据/文科跨赛（零基础友好）
- **AI 边界**：AI 主力辅助清洗/异常值/绘图；**数据规律解读、深度分析、显著性验证靠人工**
- **→ 进 C1 预测 / C2 评价 / C3 分类聚类 / C4 代价敏感 / C5 混合效应 / C6 成分数据 / C7 统计检验；预处理见 U2**

### 题型 D · 应用型优化类（对应国赛 D 题，难度适中，性价比高）
- **特征**：工业生产/物流/管理实际场景，以"最优决策"为核心，介于纯机理与纯数据之间，开放度低于 B，容错率高
- **关键词**：生产方案、成本控制、检测流程、供应链、效益最大化、风险控制
- **核心模型**：动态规划、决策树、整数规划、费用流、期望收益建模、随机/鲁棒优化、CVaR、动态定价/库存
- **适配**：全专业通用，新手/跨专业首选
- **AI 边界**：AI 辅助梳理流程、算基础方案；**决策逻辑优化、方案优劣与落地风险分析靠人工**
- **→ 进 D1 决策序贯 / D2 随机风险 / D3 经济运营优化；规划方法复用 B1，求解器规模选择见 D1 要点**

# 题型 A · 机理分析类（物理/力学/动态系统，难度最高）

## A1 微分方程类

| 方法 | 适用场景 | 关键特点 |
|------|---------|---------|
| 常微分方程 (ODE) | 动态系统建模 | 初值问题 |
| 偏微分方程 (PDE) | 时空演化问题 | 边界条件 |
| 传染病模型 SIR | 疫情传播 | 基本再生数 R0 |
| Lotka-Volterra | 捕食者-猎物 | 平衡点分析 |
| Agent-Based Model | 个体行为模拟 | 微观到宏观 |

### 微分方程求解要点

1. **解析解 + 数值解互证**：参数（阻尼/系数）**恒定**时优先求解析解——常系数线性 ODE 用特征根法/拉普拉斯变换给出闭式表达；参数**时变/非线性**时用数值解（4 阶 Runge-Kutta、`ode45`/`ode15s`）。**恒定情形下两者必须数值吻合**，这是机理题最强的求解正确性验证（如 2022A 波浪能：恒定阻尼拉普拉斯解析解 vs 变阻尼 R-K 数值解互证）。
2. **刚体/多体动力学**：转动惯量用平行移轴定理；耦合自由度（如垂荡+纵摇）建多元二阶 ODE 组联立求解。
3. **数值积分目标量**：对时间积分的性能量（平均功率等）用 trapz/Simpson，步长须做收敛性验证。
4. **检验**：解析-数值互证、能量/守恒量核对、积分步长收敛性、与 baseline 对比（详见 writing-standards.md 第 13 节）。

## A2 几何/运动学建模类 [B3 元启发式共用]

| 方法 | 适用场景 | 关键特点 |
|------|---------|---------|
| 运动学轨迹建模 | 多体相对运动（飞行器/弹体/目标） | 位置-速度-时间参数方程，统一惯性坐标系 |
| 视线遮挡/覆盖判定 | 遮蔽、通视、覆盖率问题 | 指示函数 $\mathbb{1}[\cdot]$ 判定点/线/体相交 |
| 数值积分目标量 | 目标是"持续时长/累计面积"等积分量 | 目标 $=\int \mathbb{1}[\text{条件}(t)]\,dt$，固定步长 $\Delta t$ 离散累加 |
| 空间解析几何 | 距离/夹角/相切/投影约束 | 向量法、点到线/面距离、圆锥光线 |
| 计算几何定位/交会 | 由角度/距离反求位置（无源定位、交会对接） | 轨迹（定位圆/圆弧）交点、旋转变换、向量叉积定圆心；可与代数（正弦/余弦定理）解互证 |
| 多刚体链/铰接递推 | 铰接链结构（如板凳龙把手、机械臂） | 把手→把手位置递推（余弦定理/约束传递） |
| 临界事件定位 | 求碰撞/相切/到达等临界时刻 | 该量随时间单调时用二分搜索 / ODE event detection 高效定位 |
| 排样/装箱/空间利用率 | 在有限区域内放置物体使利用率最大（装箱、版面、圈养布局） | NP-hard，启发式(BL/NFDH/最低水平线)或整数规划；蒙特卡洛随机投放估利用率 |

### 排样/空间利用率题求解要点

"有限区域内排布物体使利用率最大"（如 2023D 圈养湖羊空间利用率）：
1. **建模**：决策变量为各物体位置/朝向（连续）或占位格（0/1 整数），约束含不重叠、边界、间距；目标为利用率 $=\frac{\text{占用面积}}{\text{总面积}}$
2. **求解**：小规模用整数规划精确解；大规模用排样启发式（Bottom-Left、最低水平线）或元启发式（遗传/退火）排布
3. **利用率估计**：不规则占地用**蒙特卡洛随机投点**估面积/利用率（撒 N 个点数落入比例，$O(1/\sqrt N)$ 收敛，须报 N 收敛性）
4. **检验**：与优化前 baseline 利用率对比、不同布局参数灵敏度、约束满足验证（详见 writing-standards.md 第 13 节）

### 几何/光学物理题求解要点

物理性能由多个几何/物理子效率决定时（如 2023A 定日镜场输出功率）：
1. **乘性效率分解**：把宏观性能拆成若干物理子效率连乘，如光学效率 $\eta=\eta_{\text{阴影遮挡}}\cdot\eta_{\text{余弦}}\cdot\eta_{\text{截断}}\cdot\eta_{\text{大气透射}}\cdot\eta_{\text{反射}}$，逐项按几何/物理机理建模
2. **几何量计算**：太阳位置/光线方向用天文几何（高度角/方位角）+坐标变换；遮挡/截断用**蒙特卡洛光线追踪**或离散网格投影统计被挡比例
3. **时空积分求平均**：年/日均性能用对时间（多时刻采样）数值积分，步长/采样数须做收敛性验证
4. **优化**：场布局参数（位置/尺寸/数目/高度）用变步长搜索/网格划分/元启发式优化
5. **检验**：同几何/优化题，报收敛性、网格/采样收敛、与理论界或 baseline 对比（详见 writing-standards.md 第 13 节）

### 几何优化题求解要点

"几何场景 → 数值积分指标 → 优化"是常见无数据型范式（如遮蔽时长最大化）：
1. 在统一坐标系下写出各物体的运动学方程 $\mathbf{r}_i(t)$
2. 用指示函数表达几何条件（如视线是否被遮蔽）：$\text{IsCovered}(t)=\mathbb{1}[\cdots]$
3. 目标量为指示函数的时间积分：$T_{\text{cover}}=\int_{t_0}^{t_1}\text{IsCovered}(t)\,dt$，用固定步长 $\Delta t$ 数值积分（步长须做收敛性验证）
4. 决策变量（投放时刻/角度等）经元启发式（模拟退火/遗传）优化，含时序约束（如 $t_{i,j}\ge t_{i,j-1}+\delta$）
5. **检验不报 R²/AUC**，改报：目标收敛曲线、多初值/多次重启稳定性、与 baseline 对比、$\Delta t$ 敏感性、约束满足验证（详见 writing-standards.md 第 13 节）
6. 求"碰撞/相切/到达"等**临界时刻**时，若该量随时间单调，用二分搜索/事件检测高效定位，避免全程暴力扫描；铰接链结构（如板凳龙）用把手→把手递推传递位置

### 几何定位/反求题"双解互证"要点

由方位角/距离反求目标位置的题（如 2022B 无人机纯方位无源定位）：
1. **代数解**：极坐标分类讨论 + 正弦/余弦定理列三角方程，代数消元求解
2. **几何解（计算几何）**：将每个方位角约束转化为一条轨迹（定位圆/圆弧），多约束**轨迹交点**即目标位置；用旋转变换、向量叉积精确定圆心，避免联立高次方程
3. **双解互证**：两条独立路径（代数 vs 几何）数值一致是最强可信度证明，呼应"多方法交叉验证"主线
4. **检验**：误差分析（角度/位置误差对解的影响）、退化构型识别（共线/共圆导致不可定位）、与仿真生成样本对比（详见 writing-standards.md 第 13 节）

## A3 反问题/参数反演类

| 方法 | 适用场景 | 关键特点 |
|------|---------|---------|
| 第一性原理正问题建模 | 由物理/机理定律推导观测量 | 麦克斯韦/菲涅尔/传热/波动等控制方程 |
| 最小二乘参数反演 | 由观测反推未知参数 | $\min_\theta \lVert y_{\text{obs}}-f(\theta)\rVert^2$，非线性用 `lsqnonlin`/L-M |
| 频域/变换域反演 | 周期/振荡信号反演 | 傅里叶变换提取频率 → 反推物理量（如厚度/周期） |
| 椭偏/光谱拟合 | 薄膜厚度、折射率反演 | 多波长联合拟合，提高可辨识性 |
| 正则化反演 | 病态/欠定反问题 | Tikhonov 正则化，抑制噪声放大 |

### 反演题求解要点

1. 先建**正问题**：给定参数 $\theta$ 如何生成观测 $y=f(\theta)$（物理机理，量纲必须自洽）
2. **反演**：最小化观测残差求 $\hat\theta$；多组观测（多波长/多角度）联合提升可辨识性
3. 评估**病态性**（条件数/可辨识性），必要时正则化
4. **检验不套用分类 AUC**，改报：拟合 RMSE/$R^2$、预测曲线与实测**叠加对比图**、参数**置信区间**、对噪声的稳健性（详见 writing-standards.md 第 13 节）

# 题型 B · 综合优化类（图论/运筹/规划/调度，综合性最强）

## B1 规划类（LP/IP/MIP/NLP/多目标/目标规划/排列组合）

> 整数规划/混合整数规划在 **B 与 D 题型共用**；大规模整数规划的求解器选择见 D1「决策优化题求解要点」第 2 条。

| 方法 | 适用场景 | 关键特点 |
|------|---------|---------|
| 线性规划 (LP) | 线性目标+线性约束 | 单纯形法，成熟稳定 |
| 整数规划 (IP) | 决策变量为整数 | 分支定界法 |
| 混合整数规划 (MIP) | 部分整数部分连续 | 大规模问题 |
| 非线性规划 (NLP) | 非线性目标或约束 | 梯度下降、SQP |
| 多目标优化 | 多个目标同时优化 | Pareto前沿，NSGA-II |
| 动态规划 (DP) [亦见 D1 多阶段决策] | 多阶段决策问题 | 最优子结构 |
| 目标规划 | 多目标有优先级 | 偏差变量法 |
| 排列组合优化 | 调度、路径问题 | 遗传算法、模拟退火 |

## B2 图论/网络类

| 方法 | 适用场景 | 关键特点 |
|------|---------|---------|
| 最短路径 (Dijkstra) | 路径规划 | 非负权重 |
| 最小生成树 | 网络连接优化 | Prim/Kruskal |
| 最大流 | 网络流量优化 | Ford-Fulkerson |
| 网络流模型 | 运输、分配问题 | 线性规划转化 |
| 复杂网络分析 | 社交/交通网络 | 度分布、聚类系数 |

## B3 启发式/元启发式 [A2几何 / D 共用]

| 方法 | 适用场景 | 关键特点 |
|------|---------|---------|
| 遗传算法 (GA) | 组合优化 | 编码-选择-交叉-变异 |
| 模拟退火 (SA) | 全局优化 | 温度下降策略 |
| 蚁群算法 (ACO) | 路径优化 | 信息素更新 |
| 粒子群优化 (PSO) | 连续优化 | 速度-位置更新 |
| 禁忌搜索 (TS) | 组合优化 | 禁忌表 |
| 有信息搜索 (A*/IDA*) | 离散状态空间求最少步数/最优序列，有可设计的估价函数 | 启发函数 $f=g+h$ 引导，比盲目元启发式更快收敛、更不易陷局部解；IDA* 带阈值剪枝省内存 |

> 元启发式（GA/SA/ACO/PSO）是**盲目搜索**，靠随机性探索；当问题有**可设计的估价函数**（到目标的距离/损失，如 L2 loss）时，改用**有信息搜索 A*/IDA***（带阈值剪枝）能更高效逼近最优、避免陷入局部解（如 2022B 无人机最少调整步数：IDA* 启发 + L2 loss 估价函数，5 轮 16 架次即调到 6 位有效数字）。估价函数须满足可采纳性（不高估真实代价）才保证最优。

# 题型 C · 数据分析类（统计/挖掘/分类，选人最多）

## C1 预测类

| 方法 | 适用场景 | 关键特点 |
|------|---------|---------|
| 灰色预测 GM(1,1) | 小样本、单调趋势 | 准指数规律检验 |
| 时间序列 (ARIMA) | 平稳/差分平稳序列 | ACF/PACF定阶 |
| 指数平滑 | 短期预测、趋势平滑 | Holt-Winters |
| 回归分析 | 因果关系建模 | 多重共线性检验 |
| 广义相加模型 (GAM) | 非线性关系、平滑趋势拟合 | 样条平滑函数 $s(x)$，自动捕捉非线性，$R^2$ 显著优于刚性参数模型 |
| 神经网络 (LSTM/RNN) | 复杂非线性时序 | 大数据量需求 |
| 马尔可夫链 | 状态转移预测 | 无记忆性 |
| 灰色马尔可夫 | 小样本+状态转移 | 组合方法 |
| 小波变换 (CWT/DWT) | 非平稳信号去噪、突变点检测、多尺度趋势分离 | 时频局部化，小波基(db/sym/haar)+分解层数；优于傅里叶处理非平稳/突变 |

### 小波分析使用要点

时序/信号含噪声、突变或多尺度成分时（如 2023E 水沙序列）：
- **去噪**：小波分解 → 阈值处理高频系数 → 重构，比滑动平均更保边缘
- **突变检测**：小波系数模极大值定位突变点，与 Fisher 最优分割互证分段
- **多尺度分离**：把序列分解为趋势项+周期项+噪声，分别建模
- 小波基与分解层数须说明选择依据（信号特征匹配），实现：MATLAB `cwt`/`wavedec`、Python `pywt`

### GAM 使用要点

当标准参数模型（Logistic、指数等）拟合度 $R^2 < 0.6$ 时，优先考虑 GAM：
- 模型形式：$g(E[Y]) = \beta_0 + s_1(x_1) + s_2(x_2) + \cdots$
- $s(\cdot)$ 为平滑样条函数，自由度由数据驱动
- 优势：无需预设函数形式，自动适应数据的非线性趋势
- MATLAB 实现：`fitrgam`（R2021b+）或 Python `pygam` 库

## C2 评价类

| 方法 | 适用场景 | 关键特点 |
|------|---------|---------|
| 层次分析法 (AHP) | 多准则决策 | 一致性检验 CR<0.1 |
| 熵权法 | 客观赋权 | 信息熵 |
| TOPSIS | 多属性评价排序 | 正负理想解 |
| 模糊综合评价 | 模糊性问题 | 隶属度函数 |
| 主成分分析 (PCA) | 降维、综合评价 | 方差贡献率 |
| 数据包络分析 (DEA) | 效率评价 | 投入产出 |
| 秩和比法 | 多指标综合评价 | 非参数 |
| 灰色关联分析 (GRA) | 小样本、因素与母序列的关联排序 | 按曲线**形状相似度**算关联度，无分布假设，样本少也可用 |

### 灰色关联分析使用要点

变量少、样本小、分布未知时，比 Pearson/Spearman 更稳健地排"哪些因素影响最大"（如 2023C/E 影响因子排序）：
- 选参考序列（母序列，如目标量）与比较序列（各因素），无量纲化后算各点关联系数 $\xi_i(k)=\frac{\min\min|\cdot|+\rho\max\max|\cdot|}{|x_0(k)-x_i(k)|+\rho\max\max|\cdot|}$，分辨系数 $\rho$ 常取 0.5
- 关联度 $r_i=\frac{1}{n}\sum_k\xi_i(k)$，按 $r_i$ 排序得因素重要性；输出关联度排序条形图
- 与相关分析互补：相关看线性/单调强度，GRA 看整体曲线贴合度，适合小样本

## C3 分类/聚类类

| 方法 | 适用场景 | 关键特点 |
|------|---------|---------|
| K-means | 无监督聚类 | 需预设K |
| 层次聚类 | 树形分类 | 不需预设K |
| DBSCAN | 密度聚类 | 发现噪声点 |
| SVM | 二分类/多分类 | 核函数选择 |
| 随机森林 | 分类+特征重要性 | 抗过拟合 |
| XGBoost | 高维特征分类、特征交叉 | 自动捕捉高阶交互，通常 AUC > 0.9；内置正则化防过拟合 |
| LightGBM | 大规模高维分类 | 训练速度优于 XGBoost，适合特征数 > 50 的场景 |
| 逻辑回归 | 二分类 | 可解释性强 |
| 判别分析 | 已知类别分类 | Fisher/LDA |
| 偏最小二乘判别 (PLS-DA) | 高维强共线特征、小样本判别（成分/光谱） | 先 PLS 提取潜变量再判别，抗共线性；化学计量学标准法，优于普通 LDA（2022C 问2/3 亚类判别） |
| 有序样本聚类 (Fisher 最优分割) | 对**有序**序列（时序/空间/数值序）做连续分段 | 段内方差和最小，给最优分段；用于阶段划分、突变点、分箱 |

### 有序样本聚类 (Fisher 最优分割) 要点

普通聚类（K-means/层次）打乱顺序，**不适合**对时间/空间有序序列分段；需保持顺序连续分段时用 Fisher 最优分割（如 2023E 水沙阶段划分）：
- 目标：把 $n$ 个有序样本切成 $K$ 段，使各段**段内离差平方和之和最小**，动态规划精确求解
- **段数 $K$ 选择**：画"分段数-段内总误差"曲线取拐点（肘部法），或结合业务可解释性
- 与小波突变检测互证分段点位置；输出带分割竖线的序列分段图

### 聚类选 K 准则

K-means/层次聚类需确定簇数 $K$，**禁止拍脑袋指定**：
- **k-means++ 初始化**：避免随机初值陷入差的局部解
- **轮廓系数**（Silhouette）：取使平均轮廓系数最大的 $K$（越接近 1 越好）
- **肘部法**（Elbow）：以 SSE 随 $K$ 的拐点定 $K$
- **GMM + BIC/AIC**：簇有重叠时用高斯混合模型，按信息准则选 $K$
- 分组结果须有业务/物理可解释性（如按 BMI 分箱后各组样本量均衡）

### 集成树模型选择指南

当线性分类器（逻辑回归、LDA）AUC < 0.8 且特征维度较高时，升级路径：
1. **XGBoost**：首选，自动处理特征交叉（如 $Z_{21} \times GC_{21}$ 的非线性耦合），内置 L1/L2 正则化
2. **LightGBM**：特征数 > 50 或样本量 > 10万 时优先，训练速度快
3. **随机森林**：需要特征重要性排序时使用，可作为 baseline 对比
4. **PLS-DA**：特征**高度共线**（如成分数据各氧化物互相关、光谱相邻波段）且样本量小时，优先于普通 LDA/逻辑回归；常与 CLR/ILR 变换（见 C6「成分数据分析」）联用

## C4 代价敏感/不平衡学习

| 方法 | 适用场景 | 关键特点 |
|------|---------|---------|
| 代价敏感学习 | 假阳性/假阴性代价不对称 | 不对称代价矩阵，如 FN:FP = 100:1 |
| 阈值移动 | 筛查类问题要求高灵敏度 | 在保证 Recall ≥ 95% 下最大化 Specificity |
| SMOTE/过采样 | 正负样本严重不平衡 | 合成少数类样本 |
| 加权损失函数 | 训练时调整类别权重 | class_weight 参数 |

### 医疗筛查阈值选取原则

**禁止**直接使用 F1-score 最大化或准确率最大化确定分类阈值。正确流程：
1. 设定灵敏度硬性约束：$\text{Sensitivity} \ge 95\%$（或更高，视领域而定）
2. 在满足约束的前提下，最大化特异度 $\text{Specificity}$
3. 使用代价矩阵量化不同类型错误的临床代价
4. 输出 ROC 曲线和 PR 曲线，标注选定阈值的位置

## C5 混合效应模型

| 方法 | 适用场景 | 关键特点 |
|------|---------|---------|
| 线性混合效应模型 (LME) | 重复测量、聚类数据 | 固定效应（总体趋势）+ 随机效应（个体差异） |
| 非线性混合效应模型 (NLME) | 非线性增长 + 重复测量 | 个体随机效应极大吸收残差，$R^2$ 大幅提升 |
| 广义线性混合模型 (GLMM) | 非正态响应 + 聚类结构 | 二项/泊松分布 + 随机效应 |

### NLME 使用要点

当数据包含同一对象的多次观测（如多次采血、多次检测）时，**必须**考虑混合效应：
- 固定效应：总体趋势（如孕周对浓度的平均影响）
- 随机效应：个体差异（如每个孕妇的基线浓度和变化速率不同）
- 模型形式：$y_{ij} = f(\theta_i, x_{ij}) + \epsilon_{ij}$，其中 $\theta_i = \theta + \eta_i$，$\eta_i \sim N(0, \Omega)$
- 忽略随机效应 → 残差被个体差异淹没 → $R^2$ 极低
- MATLAB 实现：`nlmefit` / `nlmefitsa`；Python：`statsmodels.MixedLM`

## C6 成分数据分析（约束比率数据）

数据各分量受**闭合约束**（和为常数，如成分占比和为 100%、营养构成、合金/玻璃成分、投资组合权重、地球化学元素比例）时，样本落在单纯形（simplex）上而非欧氏空间。**直接对原始百分比做相关/PCA/回归会产生伪相关（spurious correlation）与负偏倚**——某一分量增大必然挤压其余分量。必须先做对数比变换映射到欧氏空间再建模（如 2022C 古代玻璃成分分析三篇均如此）。

| 方法 | 适用场景 | 关键特点 |
|------|---------|---------|
| 中心化对数比变换 (CLR) | 成分数据降维/聚类/相关分析 | $\text{clr}(x)_i=\ln\frac{x_i}{g(x)}$，$g(x)$ 为几何均值；各分量对称，但协方差阵奇异（不满秩） |
| 等距对数比变换 (ILR) | 成分数据回归/判别（需满秩协方差） | 正交基下 $D-1$ 维坐标，协方差满秩，适合做下游统计建模 |
| 加性对数比变换 (ALR) | 选定参照分量的比率分析 | $\ln(x_i/x_D)$，简单但不对称、依赖参照选择 |
| Aitchison 距离/几何 | 成分样本间距离、聚类 | 单纯形上的等距几何，替代欧氏距离做聚类/判别 |
| Dirichlet 回归 | 由协变量预测整条成分分布 | 响应为成分向量，直接建模和为 1 的占比，优于对各分量分别回归 |

### 成分数据处理要点

1. **闭合校验与预处理**：先核对各样本分量和（应≈100% 或 1，偏离过大的样本按异常剔除或归一化闭合，如 2022C 对成分和在 [85%,105%] 外的样本剔除）；零值需做**零替换**（乘性替换/加微小量），否则对数发散。
2. **变换后再建模**：CLR/ILR 变换 → 再做 PCA、聚类（配 Aitchison 距离）、相关分析、判别（PLS-DA 见 C3 分类类）；**禁止对原始百分比直接算 Pearson 相关或做主成分**。
3. **预测风化前/原始成分**：用同类未风化样本的成分分布 + 风化趋势（对数比空间回归或 Dirichlet 回归）反推，输出仍须闭合为 100%。
4. **检验**：变换前后结论一致性、零替换敏感性、成分预测的闭合性与实测对比（详见 writing-standards.md 第 13 节成分数据维度）。

## C7 统计检验工具箱 [全题型共用]

| 检验 | 用途 | 判据/备注 |
|------|------|----------|
| 正态性检验 | 拟合/残差是否正态 | QQ 图 + Shapiro-Wilk / K-S，$p>0.05$ 不拒绝正态 |
| Spearman 秩相关 | 非线性/非正态单调相关 | 优于 Pearson 处理非线性，配相关热力图 |
| Pearson 相关 | 线性相关 | 需近似正态 |
| 卡方独立性检验 (χ²) | 两个**分类变量**是否关联（列联表） | 期望频数≥5 适用；$\chi^2=\sum\frac{(O-E)^2}{E}$，用于"风化与玻璃类型/颜色/纹饰"等类别关联（2022C 问1） |
| Fisher 精确检验 | 小样本、列联表期望频数<5 | 精确概率，2022C 对组别频数小于 5 的列联表替代卡方 |
| Yates 连续性校正 | 2×2 列联表、样本偏小 | 卡方统计量减半个连续性修正，防小样本高估显著性 |
| Wilcoxon 秩和 / Mann-Whitney U | 两组非正态/有序数据的分布差异 | 非参数，替代两样本 t 检验；2022C 比较两类玻璃成分差异 |
| 多重比较校正 | 同时做多个假设检验 | **Bonferroni**（保守）/ **FDR (BH)**（高维），防假阳性膨胀 |
| 非线性最小二乘拟合优度 | 拟合模型评估 | $R^2$、RMSE、残差随机性检验 |
| 方差分析 ANOVA | 多组均值差异 | F 检验，需方差齐性 |
| 抽样方案/样本量设计 | 质量抽检：定"抽多少、判几个次品就拒收" | 由两类错误约束 + 检测成本权衡求最优 $(n,c)$ |
| R/S 分析 (Hurst 指数) | 判断时间序列的长记忆性/趋势持续性 | 重标极差求 $H$：$H>0.5$ 持续(长记忆)、$<0.5$ 反持续、$=0.5$ 随机游走 |

> 时序预测建模**前**宜先用 R/S 分析判断长记忆性：$H>0.5$ 说明趋势可持续、适合趋势外推/分形模型；$H\approx0.5$ 接近随机游走，长期预测不可靠，须谨慎（如 2023E 用 Hurst 指导预测方法选择）。

> 做相关性分析必报相关系数类型（Pearson/Spearman）与显著性；同时检验多个变量/分组时**必须**做多重比较校正（Bonferroni 或 FDR），否则假阳性率失控。

> **类别变量×类别变量**的关联（如"是否风化"与"玻璃类型/颜色/纹饰"）用**卡方独立性检验**，不能用 Pearson/Spearman（那是数值变量相关）；样本小或列联表期望频数<5 时改用 **Fisher 精确检验**或 **Yates 连续性校正**。两组非正态/有序数据比分布差异用 **Wilcoxon 秩和/Mann-Whitney U**。同一组数据做多个列联表检验时同样要做多重比较校正。

### 相关性驱动的关联/替代/互补建模

"考虑因素间相关性 / 物项间替代性·互补性"这类开放子问题（如 2024C 问三），标准套路是**先测相关、再转约束**，避免空泛定性描述：

1. **测相关**：对历史数据算 Spearman（非线性/非正态优先）或 Pearson 相关系数，配相关性热力图，识别强关联对（$|\rho|$ 大且显著）
2. **转约束/系数**：把识别出的关系**显式写入优化模型**——
   - 替代性：强正相关或同类可互换物项，用替代系数关联其决策量（一增一减），或并为同一聚合变量
   - 互补性：协同增益用互补系数修正产量/收益，或加"尽量同时选用"的软约束（惩罚项）
   - 价-量-成本关系：用供需关系或回归（销量~价格/成本）把一个变量的变化传导到另一个
3. **对比**：与不考虑相关性的基准模型对比结果，说明相关性带来的改变方向与幅度是否合理

### 抽样检测方案设计要点

"设计检测次数尽可能少的抽检方案"（如 2024B 问一）本质是**两类错误约束下的样本量优化**，三种合法框架，由严谨到简便：

1. **功效函数 ↔ 检测成本权衡（最严谨，首选）**：以"检测成本 = 单价 × n"与"假设检验功效函数"线性组合为决策函数，求使决策函数取极值的最优样本量 $n$
2. **正态近似解 $(n,c)$**：二项分布经棣莫弗-拉普拉斯定理近似正态，由 $\alpha,\beta$（两类错误上界）、标称值 $p_0$、备择 $p_1$ 联立不等式解出 $n_{\min}$ 与判据 $c$
3. **(n,c) 国标查表法**：对接 GB/T 2828 等国家标准，可信度高、可直接查表
- 三种方法所得 $n$ **量级应一致**；若相差一个数量级以上，说明建模有误（见 writing-standards.md 第 13 节数量级合理性核对）

# 题型 D · 应用型优化类（生产/物流/管理决策，性价比高）

## D1 决策优化/序贯决策类

生产决策、检测/拆解策略、序贯检验等题型（如 2024B 生产决策）的通用范式：**0/1 决策变量 + 期望收益目标函数 + 阶段/循环约束**。

| 方法 | 适用场景 | 关键特点 |
|------|---------|---------|
| 期望收益解析建模 | 决策节点少、各方案收益可闭式推导 | 用二项/几何分布等推每方案的单件期望利润，**全枚举**取最优（给真最优） |
| 多阶段决策 + 动态规划 | 流程可分阶段、状态可定义（m 工序 n 零件） | 状态-决策模型 + **记忆化存储**，精确且可推广到一般规模 |
| 决策变量 + 元启发式 | 决策维度爆炸（>20 维）、收益需仿真 | 0/1 决策向量经模拟退火/遗传/蚁群搜索，配蒙特卡洛估期望 |
| 蒙特卡洛仿真求期望 | 收益含随机性、解析期望困难 | 大量随机路径估期望利润，收敛阶 $O(1/\sqrt N)$ |
| 概率期望 + 对称性论证 | 传输/调度/排序中求最少次数、成功率，结构高度对称（如 2022D 气象报文卫星传输） | 对称性原则降复杂度 + 组合计数 + 期望推导，分类讨论证明闭式公式（如最少传输次数 $K$↔站点数 $N$ 的关系） |

### 决策优化题求解要点

1. **建模范式**：每个"是否检测/是否拆解/是否…"设 0/1 决策变量；目标函数为**期望净利润**（售价 − 购买 − 检测 − 装配 − 拆解 − 调换损失）；拆解可二次利用时设**循环**直至盈利饱和。决策变量含义、收益项须与符号表、代码逐一对应。
2. **求解器按规模选择**（本题三解法最大的方法论差异，务必先估规模再选）：
   - 决策变量 **少（≤16~20）** → **解析期望 + 全枚举**，直接给全局最优，最稳妥（首选）
   - 流程**可分阶段、状态空间可控** → **动态规划 + 记忆化**，精确且能推广到 m 工序 n 零件的一般情形
   - **大规模线性/整数规划（数百~数万变量，目标与约束基本线性或可线性化）** → **专用精确求解器**（LINGO、Gurobi、CPLEX，或 MATLAB `intlinprog`/`linprog`、Python `PuLP`+CBC/`scipy.optimize`），通常能在可接受时间内求到全局最优或带可控间隙的解——种植/排产/选址/分配等整数规划属此档，**应优先于元启发式**
   - 维度**爆炸 / 目标非线性非凸 / 收益只能仿真** → **元启发式（模拟退火/遗传/差分进化/蚁群）**，给近似最优
3. **元启发式必报最优性间隙（铁律）**：在大规模 LP/MILP 上，遗传/差分进化等元启发式常远未收敛到全局最优（典型欠优 30%~60%，足以使同一道题不同算法的结果相差数倍）。**凡用元启发式求解可线性化的规划问题，必须以 LP 松弛上界或精确求解器的解为标尺，报告最优性间隙 $\text{gap}=(\text{上界}-\text{启发式解})/\text{上界}$**；gap 过大即判未收敛，须加大种群/迭代或改用精确求解器（详见 writing-standards.md 第 13 节）。
4. **交叉验证**：小规模时用枚举/DP 的精确解去核对元启发式的近似解（数值接近即互证，见 writing-standards.md 第 13 节多方法交叉验证）。
5. **决策结果可操作化**：把最优 0/1 向量翻译成明确的文字方案（"检验零件 1、不检验零件 2、拆解所有不合格成品"），并给对应决策指标值（期望利润）；**多期规划类**（如多年份种植/排产）还须输出"地块/资源 × 年份 × 方案"的可执行方案表并配热力图（见 section-prompts.md 图表选用指南）。
6. **检验不报 R²/AUC**，按"优化/决策"类报：不同情形下决策的稳定性、关键参数（次品率/调换损失）对决策的敏感性、与不检测/全检测 baseline 的利润对比（详见 writing-standards.md 第 13 节）。

## D2 随机优化与风险决策类

参数含不确定性（销量/价格/产量/成本随机波动）的优化题，须在"收益最大化"之外**显式处理风险**，不能只对期望参数求一次确定性最优。三类范式由简到严：

| 方法 | 适用场景 | 关键特点 |
|------|---------|---------|
| 益损值矩阵 + 决策准则 | 离散少量"状态×方案"，无概率或概率粗略 | 行=自然状态、列=备选方案、元素=收益；按准则选方案 |
| 期望值随机规划（场景法） | 不确定参数分布已知/可抽样 | 蒙特卡洛生成 N 组场景 → 对每组求最优 → 取期望收益最大的方案（两阶段随机规划） |
| 均值-风险优化（VaR/CVaR） | 既要高收益又要控尾部风险 | 目标 $\max\ \mathbb{E}[\text{利润}]-\lambda\cdot\text{CVaR}_\alpha$，业界标准、最严谨 |
| 鲁棒优化 | 只知参数范围、要最坏情况最优 | $\max_x\min_{\xi\in U} f(x,\xi)$，对分布不敏感、偏保守 |

### 决策准则（益损值矩阵）

同一益损矩阵下按决策者风险态度选准则，**应同时给 ≥2 种准则结果并说明取舍**：
- **乐观准则（maximax）**：每方案取各状态最大收益，再取最大——激进
- **悲观准则（maximin / Wald）**：每方案取各状态最小收益，再取最大——保守抗风险（粮食安全等不容失败的问题常用）
- **Hurwicz 准则**：$\alpha\cdot\max+(1-\alpha)\cdot\min$，乐观系数 $\alpha\in[0,1]$ 调节激进/保守
- **等可能准则（Laplace）**：各状态等概率，取期望最大
- **最大期望准则**：已知状态概率时取期望收益最大（信息最充分时首选）

### VaR/CVaR 要点

- **VaR$_\alpha$**：置信水平 $\alpha$（如 95%）下的最大可能损失（损失分布的分位数），不反映尾部厚度
- **CVaR$_\alpha$**（条件风险价值/期望短缺）：超过 VaR 那部分损失的条件均值，$\text{CVaR}_\alpha=\frac{1}{1-\alpha}\sum_s \max(\text{VaR}-\text{利润}_s,\,0)\cdot p_s$，**凸且可线性化**，比 VaR 更稳、更易嵌入优化
- 风险厌恶系数 $\lambda$ 越大越保守；**必须做 $\lambda$ 与置信水平 $\alpha$ 的灵敏度**，并画"风险-收益有效前沿"展示权衡
- 与「U1 蒙特卡洛与不确定性量化」互补：那节量化"误差→结论可信度"，本节是"不确定性→最优决策"

### 随机规划/场景法要点

1. **场景生成**：按不确定参数分布（均匀/正态/经验分布）用蒙特卡洛抽 N 组场景，每组含一整套参数实现
2. **求解**：对每组场景用精确求解器求最优策略 → 评估各候选策略在全部 N 组场景下的收益 → 取期望（或均值-CVaR）最大者
3. **收敛性**：报告随场景数 N 增大目标值的收敛（蒙特卡洛 $O(1/\sqrt N)$），说明 N 已足够
4. **检验**：风险-收益权衡、置信水平/风险系数灵敏度、与确定性模型（用期望参数）解对比（详见 writing-standards.md 第 13 节）

## D3 经济运营优化 [B1 规划共用]

> 以下方法本质是 B1 规划方法在生产/定价/库存场景的应用，求解器规模选择见 D1 第 2 条要点。

| 方法 | 适用场景 | 关键特点 |
|------|---------|---------|
| 0/1 背包模型 | 资源/预算约束下选项目子集使价值最大 | 经典 IP/DP，监测点选址、投资组合、方案集选取 |
| 动态定价/收益管理 | 价格影响需求、定价×补货联合求最大利润 | 先估"价格-需求(弹性)"函数，再以利润为目标优化定价与订货量 |
| 服务水平约束的库存/生产安排 | 需求波动下定生产量与安全库存（如 2022E 物料生产安排） | 需求预测（指数平滑/ARIMA）+ 安全库存，在服务水平（不缺货概率）≥阈值下最小化库存/成本 |

### 动态定价/收益管理要点

价格-需求耦合的经济优化题（如 2023C 蔬菜定价补货），标准两步：
1. **估需求函数**：用历史"销量~售价(及成本)"数据回归出需求-价格关系（线性/对数/幂律，或成本加成定价下的加成率~销量关系），量化价格弹性
2. **优化定价与补货**：以周/期利润 $=\sum(\text{售价}-\text{成本})\cdot \text{需求}(\text{售价})-\text{损耗}$ 为目标，在补货量、可售品种数等约束下求最优定价与订货量；输出可执行的"品种×定价×补货量"方案表

## U1 蒙特卡洛与不确定性量化 [A/C/D 共用]

| 方法 | 适用场景 | 关键特点 |
|------|---------|---------|
| 蒙特卡洛模拟 | 期望/概率/分布难以解析 | 大量随机抽样估计，收敛阶 $O(1/\sqrt{N})$ |
| 误差/不确定性传播 | 测量误差 → 结论的影响 | 解析法（一阶 Taylor）或抽样法，输出结论置信区间 |
| Bootstrap | 统计量的置信区间 | 重抽样估计标准误 |
| 全局灵敏度分析 | 多参数交互影响 | Sobol 指数 / Morris 筛选 |
| 蒙特卡洛估几何量 | 不规则面积/体积/利用率/遮挡比例难解析 | 随机投点数落入比例，$O(1/\sqrt N)$；光线追踪求遮挡（见 A2 几何类要点） |

### 不确定性量化要点

结论（尤其建议型）应说明**输入不确定性如何传播到输出**：
- 测量误差服从已知分布（如 $c\sim N(\mu,\sigma^2)$）时，推导关键指标（如假阴性率 $\text{FNR}=\Phi(\cdot)$）随误差的变化
- 用蒙特卡洛/Bootstrap 给出结论的置信区间或鲁棒区间，而非单点
- **比例/次品率类参数**（取值在 $[0,1]$、由抽样估计）→ 标准配方是 **Beta-二项共轭先验**：先验 $\text{Beta}(\alpha,\beta)$ + 二项似然 → 后验仍为 Beta，由后验区间端点代回模型得决策范围；**必须**做先验超参数 $(\alpha,\beta)$ 的灵敏度检验，说明结论对先验选取不敏感。比 $3\sigma$ 正态近似更严谨（正态近似在 $p$ 接近 0/1 时会越界）
- 与"灵敏度分析"互补：灵敏度看参数→结果的变化方向，不确定性量化看误差→结论可信度

## 编码防错速查（落地前逐条自检）

> Coder 写完每个 solve_problemN 后、产出数值前必须逐条核对。命中即修，不放过。

### 优化/规划类
- `scipy.optimize.minimize`/`fmincon` 只做**最小化**；最大化利润/覆盖率/得分时目标函数取负，结果记录里还原为正值。
- scipy 不等式约束形式是 `fun(x) >= 0`；写容量上限 `x <= C` 必须转成 `C - x >= 0`——容量/下界/预算约束最易写反，写完代入边界点验号。
- 不要只信求解器 `success`；最优解必须**重新代入所有约束函数**，输出每条约束的值/边界/松弛量/是否活跃。
- 整数/0-1/人数/车辆数：连续松弛取整后**必须重新验证可行性**，不可行用修复启发式或分支定界，不能直接四舍五入。
- 启发式（GA/SA/PSO）：固定随机种子、≥5 次独立运行、报均值±标准差、与小规模精确解对照才可信。
- 多目标：先各自无量纲化再加权/Pareto，禁止不同量纲目标直接相加。

### 预测/回归/统计类
- 防数据泄露：先划分训练/测试，再 fit scaler/encoder；时序**不能随机打乱**，按时间切分；测试集不得参与调参。
- 标准化只在训练集 fit、transform 测试集；类别编码也在划分后做。
- 多重共线性 VIF>10 → 岭/Lasso 或删特征；残差查正态/方差齐性/自相关；外推须声明风险。
- 类别不平衡用 F1/AUC 而非 Accuracy；时序交叉验证用 TimeSeriesSplit 而非随机 KFold。

### 几何/空间类
- 有尺寸的实体**不能降维成中心点**；碰撞/遮蔽/覆盖作用于完整边界或给等价性证明。
- 旋转矩形不要用 AABB，用 OBB/SAT；坐标系与角度单位统一；离散采样近似要做采样数收敛检查。

### 图论/网络类
- 明确有向/无向；负权边不能用 Dijkstra（用 Bellman-Ford）；最大流满足每个中间节点流量守恒。
- TSP/VRP 防子回路（MTZ 或 lazy constraint）；VRP 还要查容量与时间窗。

### 机理/动力学/数值类
- 状态变量标单位；初值/边界条件明确；刚性系统用 Radau/BDF 而非默认 RK45。
- 查守恒量（SIR 的 S+I+R=N；能量系统总能不增）；步长减半结果变化<1% 才算收敛。
- 矩阵运算查条件数/奇异性，必要时正则化；蒙特卡洛≥1e4 次并报置信区间。

### 通用
- 数据读入后先查编码/列名/形状/单位/缺失/异常，列名乱码或错位时禁止继续建模。
- 结果 JSON/CSV 必须保存每问关键数值、约束检查、图表数据来源、可复现参数；不能只把一句结论写进 md。
- 物理参数/单位/符号与 methods.md 一致；代码里不得出现与定稿不一致的量。
